User Guide
==========
Contains links to information related to installing neurocaps using Pypi, Github, or Docker, in addition to information
related to the expected structure of the BIDS directory required for proper file querying, available parcellations
for spatial dimensionality reduction, logging configurations for sequential and parallel processing, and the
different output files of neurocaps.
